# Contributing to Denuncias Policiales

Thank you for your interest in contributing to the Denuncias Policiales project! We welcome contributions from everyone.

## How to Contribute

1.  **Fork the repository** to your own GitHub account.
2.  **Clone the project** to your machine.
3.  **Create a branch** locally with a succinct but descriptive name.
4.  **Commit changes** to the branch.
5.  **Push changes** to your fork.
6.  **Open a Pull Request** in our repository.

## Development Guidelines

-   **Code Style**: Please follow the existing code style for both Python (Django) and JavaScript (Node.js).
-   **Testing**: Ensure that your changes do not break existing functionality.
-   **Documentation**: Update documentation if necessary.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on the GitHub repository. Provide as much detail as possible.
